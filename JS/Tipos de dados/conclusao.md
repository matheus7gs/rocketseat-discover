# Tipos de dados

  Conforme o ECMAScript standart temos 9 tipos de dados:

    * Data types
        * Pimitive / Primitive value
        * Structural
        * Structural Primitive
      
  ## Primitivos

    * strings
    * Number
    * Boolean
    * Undefined
    * Symbol
    * BigInt

  ## Estruturais

    * Objeto
      * Array
      * Map
      * Set
      * Date
      * ...
    * Function

  ## Primitivo Estrutural / Structural Root Primitive

    * null
